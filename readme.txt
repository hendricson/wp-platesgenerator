Auto plates generator with borders, badges and styles. Usage: {platesgenerator:content}

== Description ==

<strong>PlatesGenerator</strong> is the online license/registration plates editor which can be used by license plate dealers to accept orders for customized plates. Store owner can add their custom fonts and start accepting orders immediately!

You can watch demo at [PlatesGenerator](http://platesgenerator.com)
== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/platesgenerator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Modify settings under 'Plates Generator' menu item.

== Screenshots ==

1. Payments screen, backend settings. Here you can configure the currency. Currently, PlatesGenerator does not allow to accept money on your website, but it allows to receive orders via email.
2. Badge tab of the frontend display. Here user can choose a badge to go on the custom license plate.
3. Border tab of the frontend display. Here one can select a border for the custom plate.
